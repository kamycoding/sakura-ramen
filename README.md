## 🚀 Overview

The goal of this project was to translate a complex Figma design into a fully functional, high-performance landing page. The codebase has been strictly refactored to ensure scalability, using modern CSS methodologies and semantic structure.

## ✨ Key Features

- **Pixel-Perfect Implementation:** Precise translation of the 1440px Figma design.
- **Semantic HTML5:** Leverages meaningful tags like `<article>`, `<main>`, `<header>`, and `<address>` for superior SEO and accessibility.
- **CSS Design Tokens:** Utilizes central CSS Variables for colors, typography, and spacing, ensuring a "Single Source of Truth."
- **Smooth Navigation:** Integrated smooth-scrolling via CSS for a seamless user experience across anchors.
- **Advanced UI Interactions:** Custom hover animations using CSS pseudo-elements (`::after`) for an elegant navigation feel.
- **Accessibility (A11y):** Optimized with `aria-labels` and descriptive `alt` tags for screen readers.

## 🛠 Tech Stack

- **HTML5** (Semantic Architecture)
- **CSS3** (Flexbox, CSS Variables, Transitions)
- **Google Fonts** (Raleway & Economica)

## 📂 Folder Structure

```text
sakura-ramen/
├── assets/
│   ├── icons/       # UI Icons (SVG/PNG)
│   ├── img/         # Optimized high-res images

├── styles/
│   ├── main.css     # Refactored core styles (Design Tokens)

└── index.html       # Primary entry point``

🔧 Local Setup
Clone or download the repository.

Open the folder in VS Code.

Launch index.html using the Live Server extension.

📝 Development Notes
A significant portion of the development was dedicated to Refactoring. By centralizing spacing (gap), colors, and typography into a :root scope, the project adheres to the DRY (Don't Repeat Yourself) principle. This structure makes the project exceptionally easy to scale.
```
